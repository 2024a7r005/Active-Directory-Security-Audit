# Active Directory Security Audit

## Project Description
This project is a PowerShell-based security audit tool for
Windows systems. It identifies security vulnerabilities such
as weak password policies, unauthorized admin accounts,
open network ports, active sessions, and suspicious login
attempts.

## Features
- User account enumeration
- Administrator privilege check
- Password policy analysis
- Active session monitoring
- Firewall status check
- Shared folder detection
- Open port scanning
- Failed login attempt detection
- System information gathering

## Technologies Used
- Windows PowerShell
- Windows Event Log
- Windows Registry

## How to Run
1. Open PowerShell as Administrator
2. Navigate to folder:
   cd Desktop\Active_Directory_SecurityAudit
3. Run:
   .\AD_Audit.ps1

## Output
Script displays audit results covering users, admins,
password policies, firewall, network ports, and login events.
